Working

HTML

HTML

HTML, which stands for HyperText Markup Language, is a standardized markup language used to create and structure content on the World Wide Web. It serves as the foundational building block for web pages and is used to define the structure, layout, and elements of a webpage's content. HTML allows you to present text, images, links, forms, and other multimedia elements in a structured manner that can be displayed in web browsers.

CSS

CSS

CSS, which stands for Cascading Style Sheets, is a styling language used to control the visual presentation and layout of HTML and XML documents. It allows web developers to define how the elements and content of a webpage should appear on the screen or other media types, such as print. CSS can be included directly within an HTML document using the (style) tag as shown above.

Bootstrap

Bootstrap

Bootstrap is a free and open-source CSS framework directed at responsive, mobile-first front-end web development. It contains HTML, CSS and (optionally) JavaScript-based design templates for typography, forms, buttons, navigation, and other interface components. As of April 2022, Bootstrap is the eleventh most starred project on GitHub, with over 156,000 stars.

JS

JS

JavaScript is a versatile and widely-used programming language that enables interactive and dynamic behavior on websites. It is primarily used in web development to add interactivity, validate input, manipulate the DOM (Document Object Model), create animations, and perform a wide range of other client-side tasks within web browsers.
